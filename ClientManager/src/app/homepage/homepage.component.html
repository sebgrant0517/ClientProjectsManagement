Client Project Manager

Summary

This is the client project manager developed by Sebastian Grant. This website will be used to keep track of future projects and from whom they came. You can use this site to add new clients, update current client agreements or names, and remove clients. Each client can have multiple projects, each with their own names and descriptions. You can add, remove, or update these projects as you wish.

The database of clients and projects is on AWS, where it maintains the current list. If you delete a client, all of their projects are deleted with them.

Sebastian Grant chose to develop this website as a way of keeping up with the current projects needing to be completed. It may be used for that purpose in the coming months, as the site is updated to reflect future needs.
